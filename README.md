## Overview

The Genetic Algorithm (GA) is an evolutionary optimization algorithm inspired by natural selection.  
In this project, GA is used to generate approximate solutions for the TSP by evolving a population of candidate tours over multiple generations.

### Key GA Features:
- Random initial population  
- Fitness evaluation using total route distance  
- Tournament selection  
- Order-based crossover  
- Swap mutation  
- Tracking of the best route and convergence curve
